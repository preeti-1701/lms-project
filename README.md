# LMS Project



\# 📚 LMS Project (Learning Management System)



\## 📌 Overview

This project is a \*\*Learning Management System (LMS)\*\* developed to provide an online platform for students and instructors. It allows users to access courses, watch video lectures, track progress, and download certificates after completion.



\---



\## 🎯 Features

\- 👨‍🎓 Student Registration \& Login

\- 📖 Course Enrollment

\- 🎥 Video Lectures

\- 📊 Automatic Progress Tracking

\- 🏆 Certificate Generation after course completion

\- 👨‍🏫 Admin Panel to manage courses and users



\---



\## 🛠️ Technologies Used

\- \*\*Frontend:\*\* HTML, CSS, JavaScript  

\- \*\*Backend:\*\* Python (Django Framework)  

\- \*\*Database:\*\* SQLite  



\---



\## ⚙️ How It Works

1\. Users register and log in to the system  

2\. Students enroll in available courses  

3\. They watch video lectures  

4\. Progress is updated automatically  

5\. After completing 100%, certificate is generated
